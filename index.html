Опасные моменты с препроцессорами

  • extend коварный, провоцирует дублирование
  • variables хитрые и провоцируют баги с коллизиями имен
  • mixin тоже коварный и проыоцирует дублирование свойств
  • nesting провоцирует слишком специфичные селекторы
  • нужны source maps
  • не программируйте на CSS

Правда, не надо программировать на CSS

			
 @mixin check($slides) {
   @for $i from 1 through $slides {
     input[type=radio]:nth-of-type(#{$i}):checked ~ .slider-wrapper {
       .slide:nth-of-type(#{$i}) {
         // transform: translate3d(0,0,0);
         opacity: 1;
         .caption {
           opacity: 1;
           transform: translate3d(0, 0, 0);
         }
       }
       @include slectIndexLeft($i, $slides);
       @include slectIndexRight($i, $slides);
     }
   }
 }

   $num: 1;
